# Quiz Web App 🎯

A dynamic web-based quiz application developed using Java, Servlets, HTML, CSS, and JavaScript.

## Features

* User can start the quiz
* Multiple-choice questions
* Tracks score
* Displays final result
* Interactive and user-friendly interface

## Technologies Used

* Java
* Servlet
* HTML
* CSS
* JavaScript

## Project Structure

* `src/main/java` → Java backend logic
* `controller` → Handles requests/responses
* `model` → Stores quiz data/models

## Future Improvements

* Add database integration
* Add timer for each question
* User login system
